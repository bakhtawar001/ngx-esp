# Emoji Menu

Implementation of https://github.com/scttcper/ngx-emoji-mart

Option for removing menu functionality and only showing a single emoji with `readonly` property

Option for showing the initial value of the emoji with `emoji` property, the default is set as `:package:`

An event is triggered on making a selection with a payload of `{ newValue: string }`
