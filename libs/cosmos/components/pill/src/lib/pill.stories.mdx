# Pill

## Basic Pill

This pill is for use with anchors and buttons. It is mean to be interactive.

<Canvas>
  <Story
    name="Primary"
    args={{

  >
    {Template.bind({})}
  </Story>
</Canvas>

```html
<a href="#" cosPill>
  {{ innerText }}
  <i *ngIf="_icon.length > 1" class="fa" [ngClass]="_icon"></i>
</a>
```
