## Visual States

### Idle

- 1 px G-09 Pebble horizontal rule
- Name of the collection centered

### Hover

- Checkbox
- 1 px G-09 Pebble horizontal rule
- Name of the collection centered
- Collection Action Icons
  - Remove section (does not delete products)
  - Move section up
  - Move section down
  - Collapse/Minimize section

### Editing a section title (on Hover)

- 1 px G-09 Pebble horizontal rule
- Text box containing an editable section title, centered.

### Creating a section

- Sections must be created by a distributor; new collections do not include any sections
- Distributors add sections by hovering over the space between rows of products, and clicking that space to enter a section name. All products beneath the newly created section, now reside within that section.
