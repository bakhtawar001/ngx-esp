# Toggle

Toggles allow users to turn controls on or off.

### When to use this component

Use toggles when you want to:

- Allow a binary choice, for example, on or off.

  (toggleChange)="toggleChangeEvent($event)"

## Visual Details

Toggles are used mostly for turning something on or off. That could be a setting or filter.

### Sizing

- Each toggle is 28 x 14 px

### Typography

- The text label uses Work Sans Regualr 12

### Spacing

- The horizontal space between the toggle and label is Inline 8
- The vertical spacing between each toggle is Stack 16

---

## Layout

Toggles can be used in either inline or stacked layouts.

- ### Inline

  - Use an inline layout when vertical space is limited. Spacing between each toggle is Inline 24

  ### Stacked

  - Use a stacked layout when horizontal space is limited. Spacing between each toggle is Stack 16

---

## States

### Off

This is the default state of all radio.

### On

When someone clicks/taps a radio it becomes “selected”.

### Disabled

Use this when a interaction with a radio should be blocked
